How to style HTML

Formatting HTML is simple. You can use <p> to create paragraphs or <br> to break lines within a single paragraph. You can use <b> or <strong> to make texts bold, as well as you can use <i> or <em> to make texts italic.

You can also mix these tags to make text bold italic. Another option is to use the <span> tag to assign custom styling to a block of text. For example, the text can be underlined or have different color.
